# Puppeteer meetup

Puppeteer meetup examples

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/puppeteerMeetup.git
```

Install packages

```
npm install
```

## Execution

Execute via test script

```bash
npm run <test script>
```
